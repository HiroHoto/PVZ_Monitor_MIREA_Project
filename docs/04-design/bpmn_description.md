# BPMN-диаграмма (текстовое описание)

## Участники процесса
- Оператор ПВЗ
- Супервайзер региона
- Операционный аналитик

## Последовательность шагов
1. Добавление операции → 2. Проверка пропускной способности → 3. Уведомление о перегрузке

## Альтернативные потоки
- Если пропускная способность превышена, то отправить уведомление.
