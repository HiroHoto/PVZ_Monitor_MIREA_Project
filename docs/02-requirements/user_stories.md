# User Stories

## Роль «Оператор ПВЗ»
- Как оператор ПВЗ, я хочу видеть загрузку своей точки, чтобы планировать работу.

## Роль «Супервайзер региона»
- Как супервайзер региона, я хочу видеть сводную загрузку по ПВЗ региона, чтобы координировать ресурсы.

## Роль «Операционный аналитик»
- Как операционный аналитик, я хочу экспортировать данные в CSV, чтобы проводить анализ.
- Как супервайзер региона, я хочу видеть тепловую карту загрузки ПВЗ, чтобы выявлять проблемные зоны.
- Как операционный аналитик, я хочу видеть дашборд с пороговыми индикаторами, чтобы контролировать ключевые метрики.
