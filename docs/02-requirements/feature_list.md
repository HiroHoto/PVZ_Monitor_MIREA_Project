# Feature List (MVP)

1. Мониторинг загрузки по часам/дням:
   - Отображение данных в табличном виде
   - Фильтрация по дате и ПВЗ
   - Графическое представление загрузки
2. Выявление перегрузок относительно паспортной пропускной способности ПВЗ:
   - Расчёт загрузки в реальном времени
   - История перегрузок
3. Экспорт данных в CSV:
   - Выбор периода экспорта
   - Фильтрация данных перед экспортом
   - Автоматическая генерация отчётов
4. Фильтры по дате/ПВЗ/типу операции.
5. Тепловая карта загрузки.
