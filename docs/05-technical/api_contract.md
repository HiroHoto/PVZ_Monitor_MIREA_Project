# API-контракт

Базовый URL: `http://127.0.0.1:5000`

## Эндпоинты

### GET /api/pvz
- Описание: Получение списка ПВЗ.
- Вход: query params (фильтры по региону, типу и т.д.)
- Выход: 200 OK с телом в формате JSON.

### POST /api/operations
- Описание: Добавление новой операции.
- Вход: JSON с данными операции.
- Выход: 201 Created или 400 Bad Request при ошибке валидации.

### GET /api/report/load

- Описание: Генерация отчёта о загрузке ПВЗ.
- Вход: query params (фильтры по дате, ПВЗ, типу операции).
- Выход: 200 OK с телом в формате JSON.

### GET /api/report/heatmap

- Описание: Генерация тепловой карты загрузки ПВЗ.
- Вход: query params (фильтры по дате, ПВЗ, типу операции).
- Выход: 200 OK с телом в формате JSON.

### POST /api/notifications

- Описание: Отправка уведомлений о перегрузках.
- Вход: JSON с данными для отправки уведомлений.
- Выход: 200 OK при успешной отправке.
