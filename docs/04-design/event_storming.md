# Event Storming (текстовое описание)

## События (Events)
- Операция добавлена в ПВЗ
- Перегрузка обнаружена

## Команды (Commands)
- Добавить операцию:
  - Проверка обязательных полей
  - Валидация формата данных
  - Проверка уникальности ID операции
- Уведомить о перегрузке

## Агрегаты (Aggregates)
- ПВЗ связан с событиями загрузки и перегрузки:
  - Минимальная загрузка: 0%
  - Максимальная загрузка: 150%
  - Критический порог: >120%
