# Карточка проекта

## Проблема
Существующая система не позволяет эффективно управлять загрузкой пунктов выдачи заказов (ПВЗ), что приводит к перегрузкам и простоям.

## Целевая аудитория
Клиенты маркетплейса, которые используют ПВЗ для получения заказов.

## Ценность продукта
Система позволяет сбалансировать загрузку ПВЗ и сократить очереди/простои, улучшая пользовательский опыт.

## Функциональные границы
- Что система делает:
  - Мониторинг загрузки по часам/дням.
  - Выявление перегрузок относительно паспортной пропускной способности ПВЗ.
- Что система НЕ делает:
  - Не интегрируется с реальными геосервисами/терминалами.

## Ключевые метрики успеха
- **Целевая загрузка в часы пик:** < 85%
- **Допустимая доля интервалов с перегрузкой:** ≤ 5%
- **Методика расчета загрузки:** ops_per_hour / capacity_per_hour * 100%
- **Порог критической перегрузки:** > 120%
